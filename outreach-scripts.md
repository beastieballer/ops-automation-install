## Objection Handling

### “We’ve looked at AI already.”
Makes sense. A lot of AI pitches are too broad.

This isn’t a generic AI strategy project — it’s a practical install around one specific workflow that’s already costing time.

### “We don’t want to automate sensitive stuff.”
Agreed.

The install is designed specifically to automate triage, extraction, summaries, and draft prep — not reckless autonomy around payments or sensitive communication.

### “We’re too busy right now.”
That’s usually a sign the workflow pain is real.

The point is to remove one recurring drain quickly, not create a giant project.

### “What kind of workflow do you mean?”
Usually something like:
- inbox triage
- recurring reporting
- vendor/billing follow-up prep
- job/admin status summaries
- dashboard capture + summary
