# Ops Automation Install

Standalone landing page and offer assets for the Ops Automation Install offer.

## Files
- `index.html` — landing page
- `offer-sheet.md` — client-facing offer definition
- `sales-page.md` — long-form landing copy
- `outreach-scripts.md` — outreach templates
- `discovery-call.md` — discovery call script

## Positioning
A practical service that installs one AI workflow to automate repetitive browser-based admin work while keeping human review for sensitive actions.
